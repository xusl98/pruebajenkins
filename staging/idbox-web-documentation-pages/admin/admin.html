Administración

  • Seguridad
    • Gestiona la seguridad de IDboxRT. Podrás dar de alta Corporaciones, usuarios de esas corporaciones con sus roles y permisos, asignarles a grupos de acceso para poder trabajar con lo que necesiten.

  • Perfil de usuario
    • Configura tus preferencias como usuario del sistema. Cambia tu idioma, la imagen de tu avatar, tu contraseña y mas configuraciones.

  • Adquisición de datos
    • Configuración de la adquisición de datos en el sistema. Podrás dar de alta drivers para la adquisición de datos con la configuración por defecto o la que consideres.

  • Señales
    • Gestiona las señales del sistema, sus agrupaciones y campos dinámicos mostrados en los listados de señales.

  • Funciones
    • Gestiona las funciones del sistema.

  • Fichero de configuración
    • Permite cambiar distintos parámetros de configuración de todas las secciones de la web.
