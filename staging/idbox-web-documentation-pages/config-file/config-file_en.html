Configuration file

In the configuration file you will be able to configure different parameters for the correct operation of the web. This file is located in IDbox/Clients/Web/assets/config/config-env.js. To apply the changes made to the configuration it will be necessary to refresh the browser.

  • assetsAPI: Asset api URL
  • assetsLookupdb: Lookupdb asset api URL
  • assetsCredentials: Asset API credentials

  • allowGeneralFutureDates: Defines whether a future date is allowed to be selected in all date selectors on the website.
  • graphics:
    • trend:
      • maximumNumberOfTags: Defines the maximum number of series that can be plotted on the trend graph.
      • allowZoomByWheel Enables the functionality to zoom the trend graph using the mouse wheel.
      • maxPoints: Defines the maximum number of points to be plotted on the trend graph. This value will be used for the automatic mark calculation.
      • defaultVisualizationType: Aspect of the series to be loaded by default. The available values are Line, Spline, StepLine, Area, Column y Scatter. These values will only apply to analog signals.
    • grouped:
      • maximumNumberOfTags: Defines the maximum number of series that can be represented in the cluster plot.
      • defaultVisualizationType: Aspect of the series to be loaded by default. The available values are Line, Spline, StepLine, Area, Column y Scatter. These values will only apply to analog signals.
    • realtime:
      • maximumNumberOfTags: Defines the maximum number of series that can be plotted on the real time chart.
      • defaultPointNumber: Defines the number of points to be used by default in real time plots.
      • defaultVisualizationType: Aspect of the series to be loaded by default. The available values are Line, Spline, StepLine, Area, Column y Scatter. These values will only apply to analog signals.
  • historics:
    • maximumNumberOfTags: Defines the maximum number of series that can be represented in a history document.

  • reports:
    • maximumNumberOfTags: Defines the maximum number of signals that can be included in a report document.
