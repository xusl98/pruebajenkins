Fichero de configuración

En el fichero de configuración se podrán configurar distintos parámetros para el correcto funcionamiento de la web. Este fichero se encuentra en IDbox/Clients/Web/assets/config/config-env.js. Para aplicar los cambios realizados en la configuración será necesario actualizar el navegador.

  • assetsAPI: URL de la api de activos
  • assetsLookupdb: URL de la api de activos lookupdb
  • assetsCredentials: Credenciales para la API de activos
  • kmlAPI: URL de la api de kmls de mapas

    • silent_redirect_uri:
  • allowGeneralFutureDates: Define si se permite seleccionar una fechas futuras en todos los selectores de fechas de la web.
  • graphics:
    • trend:
      • maximumNumberOfTags: Define el máximo de series que se podrán representar en la gráfica de tendencia.
      • allowZoomByWheel Habilita la funcionalidad para hacer zoom en la gráfica de tendencia usando la rueda del ratón.
      • maxPoints: Define el número máximo de puntos que se representará en la gráfica de tendencia. Este valor se usará para el cálculo de la marca automática.
      • defaultVisualizationType: Aspecto de las series que se cargará por defecto. Los valores disponibles son Line, Spline, StepLine, Area, Column y Scatter. Estos valores solo aplicarán a señales analógicas.
    • grouped:
      • maximumNumberOfTags: Define el máximo de series que se podrán representar en la gráfica de agrupados.
      • defaultVisualizationType: Aspecto de las series que se cargará por defecto. Los valores disponibles son Line, Spline, StepLine, Area, Column y Scatter. Estos valores solo aplicarán a señales analógicas.
    • realtime:
      • maximumNumberOfTags: Define el máximo de series que se podrán representar en la gráfica de tiempo real.
      • defaultPointNumber: Define el número de puntos que se usará por defecto en gráficas de tiempo real.
      • defaultVisualizationType: Aspecto de las series que se cargará por defecto. Los valores disponibles son Line, Spline, StepLine, Area, Column y Scatter. Estos valores solo aplicarán a señales analógicas.
  • historics:
    • maximumNumberOfTags: Define el máximo de series que se podrán representar en un documento de históricos.

    • maximumNumberOfTags: Define el número máximo de señales que se pueden incluir en un documento de informe.
