1. Introducción

Desde el lanzador tendremos un acceso rápido a la creación de distintos tipos de documentos presentes en IDbox junto con las señales que tengamos añadidas a favoritos. Para acceder al lanzador debemos hacer clic en el botón situado en la parte superior derecha de la barra de herramientas.

2 Lanzador

El lanzador se divide en tres partes principales: 1) Mi selección de señales: Lista de las señales que tenemos añadidas a “Mi selección”. 2) Configuración de fechas: Permite seleccionar el intervalo temporal para el que se mostrarán los datos. Se podrán escribir las fechas en el campo de texto o pulsar sobre el calendario (2) para que se abra el selector de fechas. 3) Documentos a crear: Documentos disponibles a crear. (Gráfica de dispersión y diagrama no están actualmente disponibles para su creación desde el lanzdor.), haciendo clic sobre el documento deseado se abrirá una nueva pestaña con el documento. También tenemos la posibilidad de iniciar una nueva de regla de notificaciones con la señales seleccionadas si hacemos clic en el icono
