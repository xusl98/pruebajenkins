1. Introduction

Transitions will allow us to create document animations, so that in the same document we can visualize n documents, altering the document that is currently displayed.

They can be created from the shortcuts section in others. Clicking on this option will open a new transition document.

2 Editor

The dashboard editor consists of a grid that allows you to add, delete or modify elements to a transition.

The button on the toolbar allows you to save the document.

2.1 Adding and deleting items

To add an element, click on the transitions button.

This will open the document selector. Once the desired document has been added, it will be displayed in the document grid.

To remove an element from the transitions, click on the cross of the element.

2.2 Moving elements

You can click on an element and hold down to drag it to another position. The place where the element will be positioned when the left click is released is indicated in a darker color. Also, if there are other elements in the way, they will automatically move to make room for the new element.

2.3 Configuration options

Click on the configuration button on an element and a right panel will open with the options that can be modified.

The options are as follows:

  • Animation duration: refers to how long the animation of the transicon will last.
  • Type of animation: we can choose between the following types of animation, specified in seconds.
    • None.
    • Appear.
    • Appear from below.
    • Move to the left.
    • Move to the right.
  • Screen time: indicates the time the document will be displayed on the screen, specified in seconds.

3 Viewer

When opening a transitions document or clicking on the “Preview” button in the editor toolbar. We will see the transitions document in viewer mode, with the elements we have placed and their settings.
