1. Introduction

The signal selector is a basic tool of IDboxRT, which allows us to select the signals to be analyzed. We will be able to select signals present in: signal groups and assets. It also allows the selection of functions, calculations and aliases.

2 Signal selector

The signal selector will always keep the same structure, it will present a side panel that will allow us to choose between signal hierarchy, asset hierarchy, functions, calculated signals and aliases. In the central part, the signals to the selected node are displayed.

We can filter the signals using the search engine or apply complex filters by clicking on . This will display the filtering modal window.
