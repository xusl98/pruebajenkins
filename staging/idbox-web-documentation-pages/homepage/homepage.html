1. Introducción

Tanto en la web de IDbox como en la aplicación móvil se puede visualizar en la página de inicio un documento, el cuál será configurable a nivel de usuario y grupo de seguridad.

La página de inicio será configurable desde un documento o la página de configuración específica.

2. Selección de página de inicio desde un documento

En la barra superior de cada documento se podrá pulsar el botón , siempre y cuando el documento esté guardado. Al pulsarlo se abrirá la siguiente ventana.

Se podrá seleccionar si se quiere aplicar la página de inicio para la web, la aplicación móvil o ambos. Se podrá asignar también si se quiere que cuando se habrá la página de inicio se vea en pantalla completa.

3. Selección de página de inicio desde administración

También se puede configurar la página de inicio desde la sección específica para esa funcionalidad. Para acceder a esta funcionalidad hay que abrir el panel de usuario y pulsar en botón “Página de inicio”.

Al abrirlo se podrá ver a la izquierda una previsualización del documento seleccionado, y a la derecha podremos seleccionar si queremos aplicar la configuración a la web o a la app, además de que documento queremos ver y si lo queremos ver a pantalla completa. Si cambiamos entre los tipos de aplicación cambiarán las dimensiones de la previsualización del documento.
