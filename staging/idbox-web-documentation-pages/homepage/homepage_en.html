Home Page

  • 2 Home page selection from a document
  • 3 Home page selection from administration

1. Introduction

In both the IDboxRT web and mobile application a document can be displayed on the home page, which will be configurable at user and security group level.

The home page will be configurable from a document or the specific configuration page.

2. Home page selection from a document

The button can be clicked in the top bar of each document, provided that the document is saved. The following window will open when it is clicked.

You can select whether you want to apply the home page for the web, the mobile application or both. You can also select whether you want the home page to be displayed in full screen when the home page is opened.

3. Home page selection from administration

The home page can also be configured from the specific section for that functionality. To access this functionality you should open the user panel and click on the “Home page” button.

When you open it you will see on the left a preview of the selected document, and on the right you can select whether you want to apply the settings to the web or to the app, as well as which document you want to see and if you want to see it in full screen. Switching between application types will change the dimensions of the document preview.
