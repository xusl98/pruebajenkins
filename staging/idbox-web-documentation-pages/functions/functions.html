1. Introducción

En esta sección se explicará como crear, editar o eliminar funciones.

2. Administración de funciones

Desde el panel de control se podrá acceder a la sección de “Funciones”.

Dentro de ella podremos ver un listado de todas las funciones que tengamos disponibles.

En la parte superior derecha tendremos botones para crear, editar o eliminar funciones.

Si pulsamos el botón de eliminar se eliminarán las funciones seleccionadas.

Si pulsamos sobre crear o editar se abrirá la ventana modal de edición de funciones.

3. Edición de funciones

A continuación se explicarán todos los campos mostrados en la ventana:

  • Nombre (Campo obligatorio): Será el nombre con el que se identificará a la función en los distintos listados y documentos de la aplicación.
  • Descripción: Descripción de la función.
  • Añadir señal: Permite añadir una nueva señal a la equación.
  • Equación (Campo obligatorio): Será la base de la función, donde se indique como se calcularán los valores. Para que una función sea valida solo se podrán añadir fórmulas como las especificadas mas abajo y operadores matemáticos (+, -, /, *). Si la función fuese invalida se marcará este campo en rojo y se mostrará un mensaje informativo.
  • Función: Panel donde se mostrará una previsualización de la equación escrita en el campo anterior.

En el cálculo de las funciones se tomará siempre el resultado como una señal analógica.

3.1 Fórmulas

Debajo del panel de “Función” podremos ver un listado de las fórmulas disponibles. Las cuales están divididas en dos partes que se verán mas adelante.

Al lado derecho del listado podremos ver otro panel donde se mostrará la descripción de lo que hace cada cálculo, así como una muestra de como utilizarlo. Además, con las fórmulas de IDbox se podrá ver en el apartado de “Uso” los distintos valores que puede tomar cada parámetro.

3.1.1 Fórmulas IDbox

Las fórmulas de IDbox son cálculos que se basan en operaciones sobre señales de IDbox. Para añadir una nueva fórmula de este tipo tendremos que hacer doble click y se abrirá una ventana donde tendremos que seleccionar cada parámetro. Al pulsar “Aceptar” en la ventana se añadirá el cálculo a la equación.

3.1.2 Fórmulas matemáticas

Las fórmulas matemáticas son cálculos que no requieren del uso de señales de IDbox, aunque se pueden utilizar cálculos de IDbox como parámetro para los cálculos matemáticos.

Acos(Signal([Señal2], ‘Average’, ‘StepBefore’))

Para añadir un cálculo matemático tendremos que hacer doble click y se añadirá directamente a la equación, donde tendremos que sustituir cada parámetro por el valor deseado.
