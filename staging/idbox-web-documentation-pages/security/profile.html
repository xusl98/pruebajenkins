Perfil de usuario

Podremos entrar al perfil de usuario desde el el icono de Usuario de IDbox.

En él podremos modificar los datos del usuario con el que estemos logueados actualmente, así como cambiar el grupo de seguridad por defecto o la contraseña.

Los datos que podemos modificar en el perfil son los siguientes:

  • Imagen: Podremos seleccionar una imagen para que nos aparezca arriba a la derecha en lugar del icono
  • Nombre: Nombre real del usuario.
  • Apellidos: Apellidos del usuario.
  • Usuario: Campo que indica el “username” o nombre con el que el usuario se loguea en el sistema.
  • Email: Email del usuario.
  • Cultura: Indica la cultura en el que el usuario visualizará la web.
  • Zona horaria: Zona horaria del usuario.
  • Tema: Selección del tema en el que el usuario visualizará la web.

Además podremos modificar el grupo de seguridad por defecto por otro si tenemos más de uno. Esto provocará una desconexión para reconectarse con el nuevo grupo de seguridad.

Por último podremos modificar nuestra contraseña estableciendo la antigua una vez y la nueva dos veces para confirmar.

Para guardar cualquier cambio que hagamos aquí pulsaremos el botón de guardar
