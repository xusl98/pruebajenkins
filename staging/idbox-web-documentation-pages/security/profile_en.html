User profile

We will be able to enter the user profile from the IDboxRT User icon.

In it we can modify the data of the user with which we are currently logged in, as well as change the default security group or the password.

The data that we can modify in the profile are the following:

  • Image: We will be able to select an image to appear at the top right instead of the icon .
  • First name: User’s real name.
  • Last name: Last name of the user.
  • User: Field that indicates the “username” or name with which the user is logged into the system.
  • Email: Email of the user.
  • Culture: Indicates the culture in which the user will visualize the web.
  • Time zone: Time zone of the user.
  • Theme: Selection of the theme in which the user will visualize the web.

In addition we will be able to modify the default security group for another one if we have more than one. This will cause a disconnection to reconnect with the new security group.

Finally we will be able to modify our password by setting the old one once and the new one twice to confirm.

To save any changes we make here, click on the save button .
